Experiment 3

Support Vector Machine Classifier using Linear Kernel on Social Network Ads Dataset

Aim

To write a Python program for Support Vector Machine (SVM) classifier using the Scikit-learn module and Social Network Ads dataset to classify whether a customer purchases an SUV car or not.

Algorithm

  1. Load the Social Network Ads dataset using pandas.
  2. Select Age and Estimated Salary as input features.
  3. Select Purchased as target variable.
  4. Split dataset into training (75%) and testing (25%).
  5. Perform feature scaling using StandardScaler.
  6. Create SVM classifier using Linear Kernel.
  7. Train the classifier.
  8. Predict the test data.
  9. Generate Confusion Matrix.
  10. Calculate Accuracy, Precision and Recall.
  11. Display the decision boundary.

About Dataset

The Social Network Ads dataset contains customer information used to predict whether a customer purchases an SUV based on Age and Estimated Salary.

Dataset Characteristics

  • Number of Samples: 400
  • Number of Features: 5
  • Target: Purchased (0/1)

Features

  • User ID
  • Gender
  • Age
  • Estimated Salary
  • Purchased

Python Program



import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

from matplotlib.colors import ListedColormap

from sklearn.model_selection import train_test_split

from sklearn.preprocessing import StandardScaler

from sklearn.svm import SVC

from sklearn.metrics import confusion_matrix,accuracy_score,precision_score,recall_score

dataset=pd.read_csv("datasets/Social_Network_Ads.csv")

X=dataset.iloc[:,[2,3]].values

y=dataset.iloc[:,4].values

X_train,X_test,y_train,y_test=train_test_split(
X,
y,
test_size=0.25,
random_state=0
)

sc=StandardScaler()

X_train=sc.fit_transform(X_train)

X_test=sc.transform(X_test)

classifier=SVC(kernel="linear",random_state=0)

classifier.fit(X_train,y_train)

y_pred=classifier.predict(X_test)

cm=confusion_matrix(y_test,y_pred)

accuracy=accuracy_score(y_test,y_pred)

precision=precision_score(y_test,y_pred)

recall=recall_score(y_test,y_pred)

print("Confusion Matrix")

print(cm)

print("Accuracy :",accuracy)

print("Precision :",precision)

print("Recall :",recall)

X_set,y_set=X_test,y_test

X1,X2=np.meshgrid(

np.arange(X_set[:,0].min()-1,X_set[:,0].max()+1,0.01),

np.arange(X_set[:,1].min()-1,X_set[:,1].max()+1,0.01)

)

plt.figure(figsize=(8,6))

plt.contourf(

X1,

X2,

classifier.predict(

np.array([X1.ravel(),X2.ravel()]).T

).reshape(X1.shape),

alpha=0.75,

cmap=ListedColormap(("red","green"))

)

plt.xlim(X1.min(),X1.max())

plt.ylim(X2.min(),X2.max())

for i,j in enumerate(np.unique(y_set)):

    plt.scatter(

    X_set[y_set==j,0],

    X_set[y_set==j,1],

    c=ListedColormap(("red","green"))(i),

    label=j

    )

plt.title("SVM Test Set")

plt.xlabel("Age")

plt.ylabel("Estimated Salary")

plt.legend()

plt.grid(True)

plt.savefig("static/graphs/svm.png")

plt.close()

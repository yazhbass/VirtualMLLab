Experiment 4

K Nearest Neighbors Classifier with Uniform and Distance Weights on Iris Dataset

Aim

To write a Python program that implements and visualizes the decision boundaries of a K-Nearest Neighbors (KNN) classifier on the Iris dataset using different weighting strategies such as uniform and distance weights.

Algorithm

  1. Load the Iris dataset using load_iris().
  2. Select sepal length and sepal width as input features.
  3. Extract target labels representing three Iris classes.
  4. Split the dataset into training and testing sets using train_test_split with stratify=y.
  5. Create a pipeline with StandardScaler and KNeighborsClassifier.
  6. Initialize two subplots for visualizing decision boundaries.
  7. Train KNN classifier using uniform and distance weighting strategies.
  8. Generate decision boundaries for both models.
  9. Visualize classification regions and data points.

About the Dataset

The Iris dataset consists of measurements of three different Iris flower species. The dataset contains sepal and petal measurements and is widely used for classification experiments.

Dataset Characteristics

  • Number of Instances: 150
  • Number of Classes: 3
  • Number of Attributes: 4 numeric attributes
  • Classes: Iris Setosa, Iris Versicolour, Iris Virginica

Attributes

  • Sepal Length (cm)
  • Sepal Width (cm)
  • Petal Length (cm)
  • Petal Width (cm)

Python Program



import matplotlib.pyplot as plt

from sklearn.datasets import load_iris

from sklearn.model_selection import train_test_split

from sklearn.neighbors import KNeighborsClassifier

from sklearn.pipeline import Pipeline

from sklearn.preprocessing import StandardScaler

from sklearn.inspection import DecisionBoundaryDisplay



# Load Iris Dataset

iris = load_iris(as_frame=True)



# Feature Selection

X = iris.data[
[
"sepal length (cm)",
"sepal width (cm)"
]
]


y = iris.target




# Split Dataset

X_train, X_test, y_train, y_test = train_test_split(

    X,
    y,
    stratify=y,
    random_state=0

)




# Create KNN Pipeline

clf = Pipeline(

steps=[

(
"scaler",
StandardScaler()
),


(
"knn",
KNeighborsClassifier(
n_neighbors=11
)
)

]

)



# Create Subplots

fig, axs = plt.subplots(

ncols=2,

figsize=(12,5)

)



# Train with Different Weights

for ax, weights in zip(
axs,
("uniform","distance")
):


    clf.set_params(
        knn__weights=weights
    )


    clf.fit(
        X_train,
        y_train
    )



    disp = DecisionBoundaryDisplay.from_estimator(

        clf,

        X_test,

        response_method="predict",

        plot_method="pcolormesh",

        xlabel=iris.feature_names[0],

        ylabel=iris.feature_names[1],

        shading="auto",

        alpha=0.5,

        ax=ax

    )



    scatter = disp.ax_.scatter(

        X.iloc[:,0],

        X.iloc[:,1],

        c=y,

        edgecolors="k"

    )



    disp.ax_.legend(

        scatter.legend_elements()[0],

        iris.target_names,

        loc="lower left",

        title="Classes"

    )



    disp.ax_.set_title(

    f"KNN Classification\n(k={clf[-1].n_neighbors}, weights={weights})"

    )




plt.tight_layout()



plt.savefig(

"static/graphs/knn.png",

dpi=150,

bbox_inches="tight"

)



plt.close()

Experiment 10

Gaussian Naive Bayes Classifier

Aim

To write a Python program for implementing the Gaussian Naive Bayes Classifier using the Scikit-learn library and evaluate its performance.

Algorithm

  1. Import the necessary libraries.
  2. Load the training dataset.
  3. Load the testing dataset.
  4. Read and preprocess the datasets.
  5. Separate features and labels.
  6. Create a Gaussian Naive Bayes classifier.
  7. Train the model using the training dataset.
  8. Predict the class labels of the testing dataset.
  9. Generate the Classification Report.
  10. Generate the Confusion Matrix.
  11. Display the results.

About Dataset

The dataset contains information about a person's height, weight and gender. The Gaussian Naive Bayes classifier predicts whether a person is Male or Female based on the height and weight values.

Dataset Characteristics

  • Training Dataset : person_data.txt
  • Testing Dataset : person_testset.txt
  • Features : Height, Weight
  • Target : Gender

Python Program



import numpy as np

import matplotlib.pyplot as plt

from sklearn.naive_bayes import GaussianNB

from sklearn.metrics import classification_report

from sklearn.metrics import confusion_matrix

from sklearn.metrics import ConfusionMatrixDisplay



def prepare_person_dataset(fname):

    persons=[]

    with open(fname) as fh:

        for line in fh:

            persons.append(line.strip().split())



    dataset=[]



    for person in persons:

        height_weight=(

            float(person[2]),

            float(person[3])

        )



        dataset.append(

            (

                height_weight,

                person[4]

            )

        )



    return dataset



learnset=prepare_person_dataset(

"person_data.txt"

)



testset=prepare_person_dataset(

"person_testset.txt"

)



model=GaussianNB()



w,l=zip(*learnset)

w=np.array(w)

l=np.array(l)

model.fit(w,l)

print(model)

w,l=zip(*testset)

w=np.array(w)

l=np.array(l)

predicted=model.predict(w)

print(predicted)

print(l)

print(classification_report(l,predicted))

cm=confusion_matrix(l,predicted)

disp=ConfusionMatrixDisplay(

confusion_matrix=cm,

display_labels=model.classes_

)

disp.plot()

plt.title(

"Gaussian Naive Bayes Confusion Matrix"

)

plt.savefig(

"static/graphs/naive_bayes.png"

)

plt.close()

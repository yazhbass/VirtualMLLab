Experiment 6

K-Means Clustering Algorithm on Simulated Blob Data

Aim

To write a Python program to generate a synthetic blob dataset and cluster the data using the K-Means Clustering algorithm from the Scikit-learn library.

Algorithm

  1. Import the required libraries such as Matplotlib and Scikit-learn.
  2. Create a synthetic dataset using make_blobs() with 150 samples, 2 features, and 3 cluster centers.
  3. Visualize the generated dataset.
  4. Create a KMeans object by specifying the number of clusters, initialization method, number of initializations, maximum iterations, tolerance and random state.
  5. Train the K-Means model using fit_predict().
  6. Assign every data point to its nearest cluster.
  7. Display each cluster using different colors.
  8. Display the cluster centroids.
  9. Visualize the clustered dataset.

About Dataset

The dataset is generated using make_blobs() from the Scikit-learn library. It creates synthetic Gaussian clusters that are widely used for demonstrating clustering algorithms.

Dataset Characteristics

  • Number of Samples : 150
  • Number of Features : 2
  • Number of Clusters : 3
  • Cluster Standard Deviation : 0.5
  • Random State : 0

Other Synthetic Datasets

  • make_moons() - Generates two interleaving half-moons.
  • make_circles() - Generates concentric circular data.

Python Program



import matplotlib.pyplot as plt

from sklearn.datasets import make_blobs

from sklearn.cluster import KMeans

# Generate Dataset

X, y = make_blobs(

    n_samples=150,

    n_features=2,

    centers=3,

    cluster_std=0.5,

    shuffle=True,

    random_state=0

)

# Visualize Dataset

plt.figure(figsize=(7,6))

plt.scatter(

    X[:,0],

    X[:,1],

    c='white',

    edgecolor='black',

    marker='o',

    s=50

)

plt.title("Generated Blob Dataset")

plt.xlabel("Feature 1")

plt.ylabel("Feature 2")

plt.grid(True)

plt.savefig("static/graphs/blob_dataset.png")

plt.close()

# Create KMeans Model

km = KMeans(

    n_clusters=3,

    init='random',

    n_init=10,

    max_iter=300,

    tol=1e-4,

    random_state=0

)

# Fit Model

y_km = km.fit_predict(X)

# Cluster Visualization

plt.figure(figsize=(8,6))

plt.scatter(
X[y_km==0,0],
X[y_km==0,1],
s=50,
c='lightgreen',
marker='s',
edgecolor='black',
label='Cluster 1'
)

plt.scatter(
X[y_km==1,0],
X[y_km==1,1],
s=50,
c='orange',
marker='o',
edgecolor='black',
label='Cluster 2'
)

plt.scatter(
X[y_km==2,0],
X[y_km==2,1],
s=50,
c='lightblue',
marker='v',
edgecolor='black',
label='Cluster 3'
)

plt.scatter(
km.cluster_centers_[:,0],
km.cluster_centers_[:,1],
s=250,
marker='*',
c='red',
edgecolor='black',
label='Centroids'
)

plt.legend()

plt.grid()

plt.title("K-Means Clustering on Blob Dataset")

plt.xlabel("Feature 1")

plt.ylabel("Feature 2")

plt.savefig("static/graphs/kmeans.png")

plt.close()

print("Dataset Generated Successfully")

print()

print("Number of Clusters :",km.n_clusters)

print()

print("Cluster Centers")

print(km.cluster_centers_)

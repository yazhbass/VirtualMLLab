Experiment 5

Classification of Iris Flower Dataset using Decision Tree Classifier

Aim

To write a Python program for Decision Tree Classifier using the Scikit-learn module to classify the Iris flower dataset.

Algorithm

  1. Load the Iris dataset using load_iris() from sklearn.datasets.
  2. Prepare the input features (sepal length, sepal width, petal length, petal width) and target labels.
  3. Split the dataset into training and testing sets.
  4. Create a DecisionTreeClassifier object.
  5. Train the model using the training data.
  6. Predict the testing data.
  7. Calculate the accuracy score.
  8. Generate the confusion matrix.
  9. Visualize the decision tree.

About Dataset

The Iris dataset consists of 150 samples belonging to three classes.

  • Iris Setosa
  • Iris Versicolor
  • Iris Virginica

Dataset Characteristics

  • Number of Instances:150
  • Number of Attributes:4
  • Classes:3

Attributes

  • Sepal Length
  • Sepal Width
  • Petal Length
  • Petal Width

Python Program



import matplotlib.pyplot as plt
import pandas as pd

from sklearn.datasets import load_iris

from sklearn.model_selection import train_test_split

from sklearn.tree import DecisionTreeClassifier, plot_tree

from sklearn.metrics import accuracy_score
from sklearn.metrics import confusion_matrix
from sklearn.metrics import ConfusionMatrixDisplay

iris_data = load_iris()

X = pd.DataFrame(
    iris_data.data,
    columns=iris_data.feature_names
)

y = iris_data.target

X_train, X_test, y_train, y_test = train_test_split(

    X,
    y,
    test_size=0.2,
    random_state=1

)

clf = DecisionTreeClassifier(

    random_state=1

)

clf.fit(

    X_train,
    y_train

)

y_pred = clf.predict(

    X_test

)

accuracy = accuracy_score(

    y_test,
    y_pred

)

print("Accuracy :", accuracy)

cm = confusion_matrix(

    y_test,
    y_pred

)

print("Confusion Matrix")

print(cm)

ConfusionMatrixDisplay(
    confusion_matrix=cm,
    display_labels=iris_data.target_names
).plot()

plt.title("Confusion Matrix")

plt.savefig(
    "static/graphs/iris_confusion_matrix.png",
    dpi=150,
    bbox_inches="tight"
)

plt.close()

plt.figure(figsize=(12,8))

plot_tree(

    clf,

    feature_names=iris_data.feature_names,

    class_names=iris_data.target_names,

    filled=True,

    rounded=True,

    fontsize=10

)

plt.title("Decision Tree Classifier")

plt.savefig(
    "static/graphs/iris_decision_tree.png",
    dpi=150,
    bbox_inches="tight"
)

plt.close()

print("Graphs Generated Successfully")
